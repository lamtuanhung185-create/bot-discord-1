# MixiGaming Discord Bot

A Discord bot inspired by MixiGaming, featuring AI chat, fun commands, and more.

## Features

### Quiz Game (!quiz)
Bắt đầu một câu hỏi quiz. Người dùng có 30 giây để trả lời. Nếu trả lời đúng, người chơi sẽ nhận được **1 Hộp Khô Gà Mixi** 🍗

**Cách chơi:**
1. Gõ lệnh `!quiz` để bắt đầu
2. Bot sẽ hiển thị một câu hỏi
3. Trả lời câu hỏi trong vòng 30 giây
4. Nếu đúng, bạn sẽ nhận được khô gà!
